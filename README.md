## Shopify 图片 SEO 批量命名与压缩工具

这是一个专门为 **Shopify / 独立站卖家** 设计的商品图片预处理小工具。  
通过「上传图片 → 设置关键词 → 一键处理 → 按需下载」的流程，帮你在上架前快速完成图片重命名与压缩，让店铺页面加载更快、结构更清晰。

---

## 功能简介

- **拖拽 / 点击上传图片**
  - 支持 `JPG / PNG / WEBP / HEIC`
  - 单次最多 10 张，避免浏览器卡死
  - 所有处理 **仅在浏览器本地完成**，不会上传到服务器

- **关键词驱动的批量重命名**
  - 输入一个英文或拼音关键词（例如：`summer silk dress`）
  - 自动清洗空格与特殊字符，使用中划线连接
  - 自动生成有序文件名：`keyword-01.jpg`、`keyword-02.png` ...
  - 保留原始图片格式，不强制改成 WebP，兼容 Shopify 自带的图片优化

- **在线压缩 & 体积对比**
  - 在浏览器内使用 Canvas 对图片进行压缩
  - 结果表格中展示：
    - 压缩前体积（MB）
    - 压缩后体积（MB）
    - 节省百分比（%）
  - 若压缩后文件反而更大，会自动退回使用原图，避免“越压越大”

- **结果总览 & 单张下载**
  - 表格中展示每张图片的：
    - 预览缩略图与序号
    - 原始文件名
    - 新文件名
    - 压缩前 / 压缩后体积 & 节省比例
  - 每一行提供「下载图片」按钮，可逐张下载处理后的图片
  - 支持一键「清空」重置列表与关键词，方便分批处理不同商品

---

## 适用场景

- Shopify 新品上架前，对商品图做统一命名与压缩
- 老品重构图片资产，提高整站加载速度与视觉一致性
- 跨境卖家为英文市场准备更友好的图片文件名

## 本地开发与使用

1. **安装依赖**

```bash
npm install
```

2. **启动本地开发服务**

```bash
npm run dev
```

启动后在浏览器访问：

```text
http://localhost:5173    # 或 Vite 终端中提示的本地地址
```

3. **使用步骤**

1）在顶部区域拖拽或选择图片上传  
2）确认下方「已导入图片」中的缩略图和原始文件名  
3）在「图片名称关键词」中输入本次批量命名的关键词  
4）点击「一键重命名并压缩」  
5）在结果表格中查看压缩前后体积、节省比例，并使用「下载图片」逐张保存

---

## 构建生产版本

```bash
npm run build
```

---

## 说明

- 本工具目前只做 **命名与体积优化**，不改变图片内容（例如抠图、换背景等）。
- 所有处理逻辑运行在浏览器中，不会把你的商品图片上传到任何后端服务。
